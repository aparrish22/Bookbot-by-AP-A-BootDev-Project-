# Bookbot Project by AP!

Sup! This simple python project accepts any .txt file and displays how many words
and characters were in the text.
BookBot is my first [Boot.dev](https://www.boot.dev) project!

[Try out BookBot!](#instructions)<br>
[Checkout my Review!](#review)<br>

## Instructions:

Completed a simple python project that you can run on your own local machine.
Download any free text from [Project Gutenburg](https://www.gutenberg.org/)
and insert them into a .txt file under '/bookbot/books' directory.

Make sure python3 is installed on your local machine
(check with python3 --version).

Try out bookbot for yourself using the command: python
